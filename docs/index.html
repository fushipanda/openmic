A privacy-first CLI for capturing and structuring spoken thought. Transcription runs entirely on your machine — audio never leaves it.

$ curl -fsSL https://raw.githubusercontent.com/fushipanda/openmic/main/install.sh | bash

Python 3.12+ · GPU optional (CPU fallback) · local or any cloud LLM

what it does# 04
01
Local transcription

faster-whisper with GPU acceleration and voice-activity gating. On-device, never uploaded.

02
RAG query

ask natural-language questions across every session you've recorded.

03
Structured notes

summaries, decisions, and action items generated on demand.

04
Zero cloud calls

fully offline with Ollama, or bring any LLM provider you prefer.

privacy model

Transcription always runs locally — audio never leaves your machine, whichever LLM you pick. The only thing that's configurable is what touches the cloud afterwards.

●
Local + cloud LLM

faster-whisper on-device · Anthropic, OpenAI, Gemini or OpenRouter for query & notes · audio never uploaded.

●
Fully local

faster-whisper on-device · Ollama for everything · zero cloud calls at all.
